Celery Project
==============

> ***A python3 celery project***

Quickstart
----------

More about each of these steps can be found in the [developer guides](DEVELOPMENT.md).

- Run `asdf install`.
- Run `bin/install`.
- Run `bin/setup`.
- Fill out local environment `.env` file values.
